Steps to code in python online

  • To code in python online, visit quantumcomputing.com
  • Sign up or log in to an account.
  • Now go to the dashboard and then to projects.
  • Click the create project button on the top - right corner.
  • Give a name to your project and click create project.
  • Click on add file.

    • A file is created. To run a peice of code, click on the "run" button on the top right corner.
